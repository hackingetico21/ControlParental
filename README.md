# ControlParental
Script en PowerShell que permite crear 1 tarea en un equipo Windows, la tarea es levantar un servidor web, por medio del cual se puede controlar la maquina localmente y reiniciarla o apagarla a gusto, si junta este script con el shutdownmanager.ps1 que se encuentra en otro repositorio, armas un completo control tanto de horario como de acciones.

# Abrir PowerShell como Administrador y ejecutar:

    powershell -ExecutionPolicy Bypass -File "C:\InstalarControlPC.ps1"
